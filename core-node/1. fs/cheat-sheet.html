📁 Node.js fs Module Cheat Sheet

Modern async/await style (fs/promises) — Node.js 18+

💡 Golden Rule: Always import from node:fs/promises for modern code. Use Sync methods only during server startup. Use node: prefix to avoid npm package name collisions.
📖 Reading Files
Read

readFile

fs.readFile(path, encoding?)SYNC

Read entire file into memory. Returns string if encoding provided, otherwise Buffer.

const fs = require('node:fs/promises');

// As string
const text = await fs.readFile('./file.txt', 'utf-8');

// As Buffer (binary)
const buf = await fs.readFile('./image.png');
⚠️ Don't use on files > few MB — use streams instead.
Read

createReadStream

fs.createReadStream(path, options)

Stream file data chunk-by-chunk. Memory-efficient for large files.

const fs = require('node:fs');

const stream = fs.createReadStream('./huge.log', 'utf-8');
stream.on('data', chunk => console.log(chunk));
stream.on('end', () => console.log('Done'));
Read

readdir

fs.readdir(path, options?)SYNC

List directory contents. Returns array of strings or Dirent objects.

// Simple list
const files = await fs.readdir('./uploads');

// With file type info
const entries = await fs.readdir('./uploads', { withFileTypes: true });
for (const e of entries) {
  console.log(e.name, e.isDirectory() ? 'DIR' : 'FILE');
}
✏️ Writing Files
Write

writeFile

fs.writeFile(path, data, options?)SYNC

Write data to file. Creates file if missing, overwrites if exists.

await fs.writeFile('./out.txt', 'Hello World', 'utf-8');

// With options
await fs.writeFile('./out.txt', data, {
  encoding: 'utf-8',
  flag: 'wx'   // fail if file exists
});
⚠️ Parent directory must exist or throws ENOENT.
Write

appendFile

fs.appendFile(path, data, options?)SYNC

Append data to end of file. Creates file if it doesn't exist.

await fs.appendFile('./log.txt',
  `${new Date().toISOString()} - Event\n`);
Write

createWriteStream

fs.createWriteStream(path, options)

Writable stream for file output. Great for piping or incremental writes.

const ws = fs.createWriteStream('./big.txt');
ws.write('chunk 1');
ws.write('chunk 2');
ws.end('final chunk');
📂 Directories
Dir

mkdir

fs.mkdir(path, options?)SYNC

Create directory. Use recursive: true to create nested paths.

// Create nested dirs (mkdir -p)
await fs.mkdir('./data/2026/july', { recursive: true });

// Returns first created dir path (if recursive)
const created = await fs.mkdir('./a/b/c', { recursive: true });
// created === '/project/a' (first created)
Dir

rm / rmdir

fs.rm(path, options?)SYNC

Remove files and directories. rm is the modern universal delete.

// Delete file
await fs.rm('./old.txt');

// Delete directory + everything inside
await fs.rm('./temp', { recursive: true, force: true });

// rmdir only removes EMPTY directories
await fs.rmdir('./empty-folder');
🔥 Prefer fs.rm() over rmdir() for recursive deletes.
ℹ️ Metadata & Checks
Meta

stat / lstat

fs.stat(path)SYNC

Get file/directory metadata. lstat doesn't follow symlinks.

const s = await fs.stat('./file.txt');

s.size              // file size in bytes
s.isFile()          // true if file
s.isDirectory()   // true if directory
s.isSymbolicLink() // true if symlink
s.mtime             // last modified (Date)
s.birthtime         // created (Date)
Meta

access

fs.access(path, mode?)SYNC

Check if file exists and/or has permissions. Use instead of deprecated exists.

import { constants } from 'node:fs';

try {
  await fs.access('./file.txt');
  // file exists
} catch { /* doesn't exist */ }

// Check specific permission
await fs.access('./file.txt', constants.R_OK | constants.W_OK);
🔄 Copy, Move, Rename
Write

copyFile

fs.copyFile(src, dest, mode?)SYNC

Copy file from src to dest. Optionally control overwrite behavior.

await fs.copyFile('./a.txt', './backup/a.txt');

// COPYFILE_EXCL: fail if dest exists
await fs.copyFile('./a.txt', './b.txt', fs.constants.COPYFILE_EXCL);
Write

rename

fs.rename(oldPath, newPath)SYNC

Rename or move file/directory. Can move across directories.

// Rename
await fs.rename('./old.txt', './new.txt');

// Move to different directory
await fs.rename('./temp/file.txt', './archive/file.txt');
⚠️ Destination directory must exist or throws ENOENT.
👁️ Watching
Watch

watch / watchFile

fs.watch(path, options?, listener)

Watch file or directory for changes. Returns FSWatcher object.

const watcher = fs.watch('./config.json', (event, filename) => {
  console.log(event, filename);  // 'change' | 'rename'
});

// Watch entire directory recursively
const dir = fs.watch('./src', { recursive: true }, (event, file) => {
  console.log(file, event);
});

// Stop watching
watcher.close();
⚠️ fs.watch is fast but platform-dependent. For production, use chokidar npm package.
🔗 Symlinks & Advanced
Link

symlink / readlink / realpath

fs.symlink(target, path)SYNC

Create symlinks, read symlink targets, resolve to absolute real paths.

// Create symlink
await fs.symlink('./real.txt', './link.txt');

// Read where symlink points
const target = await fs.readlink('./link.txt');

// Resolve to absolute real path (follows symlinks)
const real = await fs.realpath('./link.txt');
Adv

chmod / chown / utimes

fs.chmod(path, mode)SYNC

Change file permissions, ownership, and timestamps.

import { constants } from 'node:fs';

// chmod 644 (rw-r--r--)
await fs.chmod('./file.txt', constants.S_IRUSR | constants.S_IWUSR |
  constants.S_IRGRP | constants.S_IROTH);

// Or use octal
await fs.chmod('./file.txt', 0o644);

// Touch timestamps
await fs.utimes('./file.txt', Date.now()/1000, Date.now()/1000);
🚀 Pro Pattern — Safe Write with Auto-Directory: 
import fs from 'node:fs/promises';
import path from 'node:path';

async function safeWrite(filePath, data) {
  await fs.mkdir(path.dirname(filePath), { recursive: true });
  await fs.writeFile(filePath, data);
}

📝 Flag Quick Reference

FlagDescription
'r'Read (default). Fails if not exists.
'r+'Read & write. Fails if not exists.
'w'Write. Creates or truncates.
'w+'Read & write. Creates or truncates.
'a'Append. Creates if missing.
'a+'Read & append. Creates if missing.
'wx'Write, fails if exists.
'ax'Append, fails if exists.
